# BOJ 1260 - DFS와 BFS

## 문제 분석
- 하나의 그래프가 주어졌을 때, DFS(깊이 우선 탐색)와 BFS(너비 우선 탐색)로 각각 탐색한 결과를 출력하는 문제입니다.
- 방문 정점이 여러 개인 경우 **번호가 작은 것부터 방문**해야 하는 제약 조건이 있습니다.

## 접근 방법
- **그래프 표현**: 인접 리스트(`adj`)를 활용하여 공간을 효율적으로 사용합니다.
- **정렬**: 각 정점의 인접 리스트를 오름차순으로 정렬하여 번호가 작은 정점을 먼저 방문하도록 합니다.
- **DFS (재귀)**: 재귀 호출을 통해 깊게 탐색하며 방문한 정점을 기록합니다.
- **BFS (Queue)**: 큐 자료구조를 사용하여 너비 방향으로 탐색하며 방문한 정점을 기록합니다.

## 코드 설명
- `adj[i].sort()`를 통해 문제의 조건인 "작은 번호 우선 방문"을 보장합니다.
- 시간 복잡도: $O(V + E)$ (V는 정점 수, E는 간선 수).
