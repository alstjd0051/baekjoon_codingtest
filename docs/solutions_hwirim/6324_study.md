# BOJ 6324 - URLs

## 문제 분석
- 주어진 URL 문자열을 프로토콜, 호스트, 포트, 경로의 네 가지 구성 요소로 파싱하는 문제입니다.
- 구조: `<protocol>://<host>[:<port>][/<path>]`
- 포트나 경로가 생략된 경우 `<default>`를 출력해야 합니다.

## 접근 방법
- **문자열 분리 (Split)**:
    - 가장 먼저 `://`를 기준으로 프로토콜과 나머지 부분을 분리합니다.
    - 나머지 부분에서 첫 번째 `/`를 기준으로 호스트/포트 영역과 경로 영역을 분리합니다.
    - 호스트/포트 영역에서 `:`를 기준으로 호스트와 포트를 분리합니다.
- **예외 처리**:
    - 분리할 구분자가 없는 경우 해당 요소를 `<default>`로 설정합니다.

## 코드 설명
- `split(delimiter, 1)`을 사용하여 첫 번째 구분자만 기준으로 나누는 방식을 활용했습니다.
- 각 테스트 케이스마다 정해진 포맷에 맞춰 출력하며, 마지막에 빈 줄을 추가합니다.
