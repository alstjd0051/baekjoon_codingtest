# BOJ 9251 - LCS

## 문제 분석
- LCS(Longest Common Subsequence, 최장 공통 부분 수열) 문제는 두 수열이 주어졌을 때, 모두의 부분 수열이 되는 수열 중 가장 긴 것을 찾는 문제입니다.
- 수열의 길이는 최대 1,000입니다.

## 접근 방법
- **동적 계획법 (Dynamic Programming)**:
    - $dp[i][j]$를 첫 번째 문자열의 $i$번째 문자까지와 두 번째 문자열의 $j$번째 문자까지의 LCS 길이라고 정의합니다.
    - 문자열이 같으면: $dp[i][j] = dp[i-1][j-1] + 1$
    - 문자열이 다르면: $dp[i][j] = \max(dp[i-1][j], dp[i][j-1])$
- 공간 최적화: 1차원 배열을 사용하여 $O(N)$ 공간 복잡도로 해결 가능합니다.

## 코드 설명
- 문자열의 길이를 $n, m$이라 할 때, $O(n \times m)$ 시간 복잡도로 해결합니다.
- 1차원 `dp` 배열을 사용하여 이전 행의 값을 `prev` 변수로 관리하며 업데이트합니다.
