# BOJ 2644 - 촌수계산

## 문제 분석
- 부모-자식 간의 관계를 1촌으로 정의하고, 주어진 두 사람 사이의 촌수를 계산하는 문제입니다.
- 가중치가 1인 무방향 그래프에서 두 노드 간의 최단 거리를 찾는 것과 같습니다.

## 접근 방법
- **그래프 구성**: 인접 리스트를 사용하여 관계를 양방향으로 저장합니다.
- **BFS 탐색**: 시작점부터 탐색을 시작하여 목표 정점에 도달할 때까지의 레벨(거리)을 큐에 저장하며 진행합니다.
- **방문 체크**: 이미 방문한 노드는 다시 방문하지 않도록 처리합니다.

## 코드 설명
- 큐에 `(노드, 거리)` 쌍을 넣어 목표 노드를 찾으면 거리를 즉시 반환하고 종료합니다.
- 탐색을 마쳤음에도 목표 노드를 찾지 못하면 `-1`을 출력합니다.
