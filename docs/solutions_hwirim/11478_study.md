# BOJ 11478 - 서로 다른 부분 문자열의 개수

## 문제 분석
- 주어진 문자열에서 가능한 모든 서로 다른 부분 문자열의 개수를 구하는 문제입니다.
- 문자열의 길이는 최대 1,000입니다.

## 접근 방법
- **Set(집합) 활용**:
    - 모든 부분 문자열을 생성하여 집합에 넣으면 중복이 자동으로 제거됩니다.
    - 문자열의 길이가 $N$일 때, 부분 문자열의 총 개수는 $N(N+1)/2$입니다.
    - $N=1000$일 때 약 50만 개의 부분 문자열이 생성되며, 이는 시간 제한(1초) 내에 충분히 처리 가능합니다.
- 시간 복잡도: $O(N^2)$.

## 코드 설명
- 이중 반복문을 사용하여 모든 시작점과 끝점에 대해 부분 문자열을 슬라이싱하여 `set`에 추가합니다.
- 최종적으로 `set`의 길이를 출력합니다.
