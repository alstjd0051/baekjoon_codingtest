# BOJ 20529 - 가장 가까운 세 사람의 심리적 거리

## 문제 분석
- MBTI 유형 16가지가 주어졌을 때, 세 사람 사이의 심리적 거리의 합이 최소인 경우를 찾는 문제입니다.
- 심리적 거리 = (A,B) + (B,C) + (A,C) 거리의 합.
- $N$의 범위가 최대 100,000으로 매우 큽니다.

## 접근 방법
- **비둘기집의 원리**: MBTI 유형은 총 16가지입니다. 만약 사람이 33명 이상이라면, 최소 3명은 같은 MBTI를 가지게 됩니다 (16 * 2 + 1 = 33). 이 경우 세 사람의 거리는 0이 됩니다.
- $N > 32$인 경우 정답은 무조건 0입니다.
- $N \le 32$인 경우 삼중 반복문을 통해 모든 조합을 확인해도 충분히 시간 내에 해결 가능합니다 ($32^3 = 32,768$).

## 코드 설명
- $N > 32$ 체크를 통해 불필요한 연산을 방지합니다.
- 거리를 계산하는 `get_dist` 함수를 활용하여 세 사람의 거리 합을 구합니다.
- 시간 복잡도: $O(T \times \min(N, 33)^3)$.
