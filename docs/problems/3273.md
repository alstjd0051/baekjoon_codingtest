# BOJ 3273 - 3273번: 두 수의 합

- 날짜: `2026-04-25`
- 문제 번호: `3273`
- 원문 링크: https://www.acmicpc.net/problem/3273

## 문제 설명

n개의 서로 다른 양의 정수 a 1 , a 2 , ..., a n 으로 이루어진 수열이 있다. a i 의 값은 1보다 크거나 같고, 1000000보다 작거나 같은 자연수이다. 자연수 x가 주어졌을 때, a i + a j = x (1 ≤ i < j ≤ n)을 만족하는 (a i , a j )쌍의 수를 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 수열의 크기 n이 주어진다. 다음 줄에는 수열에 포함되는 수가 주어진다. 셋째 줄에는 x가 주어진다. (1 ≤ n ≤ 100000, 1 ≤ x ≤ 2000000)

## 출력

문제의 조건을 만족하는 쌍의 개수를 출력한다.
