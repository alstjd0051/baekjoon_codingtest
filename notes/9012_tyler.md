# 9012 풀이

## 핵심 아이디어

스택 대신 정수 카운터 `c` 하나로 괄호 균형을 관리한다.

- `'('`를 만나면 `c += 1`
- `')'`를 만나면 `c -= 1`

올바른 괄호 문자열(VPS)이 되려면 아래 두 조건을 모두 만족해야 한다.

1. 문자열을 왼쪽부터 읽는 동안 `c`가 한 번도 음수가 되면 안 된다.
2. 문자열을 끝까지 읽은 뒤 `c == 0`이어야 한다.

## 코드 해석

```python
for _ in[0]*int(input()):
 c=0
 for i in input():
  c+=1-2*(i>'(')
  if c<0:break
 print('NO'if c else'YES')
```

- `for _ in [0] * int(input())`  
  첫 줄 입력 `T`만큼 테스트 케이스를 반복한다.

- `c = 0`  
  현재까지의 괄호 균형값을 저장한다.

- `c += 1 - 2*(i > '(')`  
  압축된 증감 표현이다.
  - `i == '('` 이면 `(i > '(')`는 `False(0)` -> `+1`
  - `i == ')'` 이면 `(i > '(')`는 `True(1)` -> `-1`

- `if c < 0: break`  
  닫는 괄호가 먼저 많아진 순간이므로 VPS가 될 수 없다.

- `print('NO' if c else 'YES')`  
  최종 `c == 0`이면 `YES`, 아니면 `NO`를 출력한다.
