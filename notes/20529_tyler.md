# 20529 풀이

## 아이디어

두 MBTI 유형의 심리적 거리는 4개 척도 중 서로 다른 척도의 개수(= 다른 글자 위치 개수)이다.

세 학생 $A,B,C$에 대한 심리적 거리는

- $d(A,B) + d(B,C) + d(A,C)$

인데, 중요한 점은 입력에 등장할 수 있는 MBTI 유형이 총 16종뿐이라는 것이다.

따라서 각 테스트 케이스마다:

1. 16개 MBTI 타입 각각의 등장 개수를 센다.
2. 타입 3개를 조합으로 고른다(중복 허용, $i \le j \le k$).
3. 해당 조합이 실제로 뽑을 수 있는지(개수 조건) 확인한다.
4. 가능한 조합 중 위 식의 합이 최소인 값을 답으로 낸다.

타입 16개에 대해 가능한 3개 조합 수는 최대 약 816개라 충분히 빠르다.

## 코드 해석

- `dist[i][j]`: 타입 `i`와 `j` 사이의 거리(다른 글자 위치 개수)
- `cnt[x]`: 타입 `x`의 학생 수
- `i==j==k`, `i==j`, `j==k`, `세 타입 모두 다름` 경우별로 필요한 개수 조건을 검사
- 가능한 조합의 $d(i,j)+d(j,k)+d(i,k)$ 중 최솟값이 정답
