# 14425 풀이

## 아이디어

집합 `S`에 문자열이 포함되는지 여러 번 확인하는 문제이므로, 파이썬 `set`을 사용하면 된다.

- `set`의 포함 여부 조회는 평균 `O(1)`
- 검사 문자열 `M`개를 순회하면서 `S`에 있으면 카운트

전체 시간 복잡도는 입력을 제외하면 `O(N + M)`이다.

## 코드 해석

- 처음 `n`개 문자열을 집합 `s`로 만든다.
- 다음 `m`개 문자열 각각에 대해 `s` 포함 여부를 검사한다.
- `True`/`False`가 `1`/`0`으로 합산되는 점을 이용해 정답을 한 줄로 출력한다.
